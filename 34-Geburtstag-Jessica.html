# Geburtstagskarte für Jessica Eine responsive Geburtstagskarte als einzelne HTML-Datei für Jessicas 34. Geburtstag. ## Datei - `34-Geburtstag-Jessica.html` - komplette Geburtstagskarte mit Layout, Animationen, Konfetti, Ballons und Musik - `favicon.svg` - kleines Icon für Browser-Tab und Lesezeichen ## Nutzung Die Datei kann direkt im Browser geöffnet werden: ```bash firefox 34-Geburtstag-Jessica.html ``` Alternativ per Doppelklick im Dateimanager öffnen. ## Funktionen - responsive Design für Smartphone, Tablet und Desktop - animierter Farbverlauf im Hintergrund - schwebende Ballons - Konfetti-Animation beim Laden und per Button - Musik-Button mit synthetischer Geburtstagsmelodie über die Web Audio API - keine externen Abhängigkeiten und keine zusätzlichen Dateien erforderlich ## Anpassen Die wichtigsten Texte stehen direkt im HTML im Bereich: ```html
``` Farben, Größen und Schatten sind im CSS über Variablen in `:root` gebündelt. Die Musiknoten stehen im JavaScript-Array `birthdaySong`. Tempo und Lautstärke können über diese Konstanten angepasst werden: ```js const MUSIC_TEMPO = 0.38; const MUSIC_VOLUME = 0.08; ``` ## Hinweis zur Musik Browser starten Audio normalerweise erst nach einer Nutzeraktion. Deshalb beginnt die Musik erst nach Klick auf `Musik starten`.
